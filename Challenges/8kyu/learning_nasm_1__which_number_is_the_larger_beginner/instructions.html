Assembly language is fun. But you need to learn it as you would need to with any other language. NASM is a fun and easy to learn variant of ASM. If you are interested in writing your first NASM programs, you can use this Kata as an entrypoint.

The goal of this Kata is very, very easy to accomplish in other programming languages and it is in NASM as well. It's whole purpose is to give you a feeling on how to write a program in NASM to complete Katas for it on this site.

Your task is to write a function, that will take two arguments of type integer and returns the one that is the largest. If both numbers are equal, return the equal number. The numbers will never be negative and will not be greater than 2000.

What you need to know:

The first argument is stored in rdi/edi.

The second argument is stored in rsi/esi.

The return value needs to be stored in rax/eax.

To tell the program to exit and return the value stored in rax/eax you need to write ret

Have fun coding :)
