Task conditions

  • Node v18.x

Goal

Create a function which formats a duration in seconds, in a human-friendly way

  • Expression is made of components
  • Component is an integer and valid unit of time, separated by a space e.g. 5 minutes
  • Unit of time is used in plural if the integer is greater than 1
  • Components have different unit of times
  • Components are separated by ", " but last component is separated by " and "
  • Components with more significant units occur before least significant one e.g. 1 second and 1 year is NOT correct
  • Component will not appear if its value is 0 e.g. 1 minute and 0 seconds is NOT correct
  • Duration specified by of a component cannot be greater than any valid unit of time e.g. 61 seconds is NOT correct

Starter code:

function formatDuration (seconds) {
  // Complete this function
}

Solution

Missing * between a and b
