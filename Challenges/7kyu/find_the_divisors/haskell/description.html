Task conditions

  • GHC 9.2.5

Goal

Create function that for given natural integer n returns list of divisors of n or string "n is prime"

Starter code:

module Divisors where

divisors :: (Show a, Integral a) => a -> Either String [a]
divisors a = error "todo: implement divisors"

Solution

Factors of n are numbers x from 2 to n - 1 that leaves no remainder when divinding n i.e. mod n x == 0

Steps:

1. Calculate factors of n
2. If there are no factors of n return string "n is prime", otherwise return factors
